# 流程图

  • 语法解释: graph 表示声明一张流程图 TD 表示方向 T (top)-D (down) 由上至下 () 表示圆形框 {} 表示菱形框
# 时序图
  • 语法解释: ->>代表实线箭头 -->>代表虚线箭头
# 状态图
  • 语法解释: [*]代表开始或结束 如果在箭头右边则表示结束
# 类图
  • 语法解释: <|-- 表示继承 + 表示 public - 表示 private
# 饼图
  • 语法解释：饼图使用 pie 表示 标题下面分别是区域名称及其所占百分比
