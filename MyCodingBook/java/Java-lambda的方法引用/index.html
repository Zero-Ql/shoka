# 方法引用

方法引用有四种引用方法

类型 语法 对应的 lambda 表达式
对象引用非静态方法 对象名 : : 实例方法名 （ i ) -> 对象名。实例方法名 (i)
类引用静态方法 类名 : : 静态方法名 引用 (i) -> 类名。静态方法名 ( i )
类引用非静态方法 类名 : : 实例方法名 引用 (i) -> 类名。实例方法名 ( i )
类引用构造方法 类名 : : new // 构造引用 (i) -> new 类名 ( i )
  • 千万注意，要引用的方法的形参和返回值一定要跟接口的一样，要不然找不到这个方法的
