# HTML 基础

# HTML 标题

HTML 的标题 (Heading) 是通过 <h1> - <h6> 标签来定义的。

<h1> 定义最大标题， <h6> 定义最小标题。

注：浏览器会自动在标题的前后添加空行。

请确保将 HTML 标题 标签只用于标题。而不仅仅只是为了生成 粗体 或 大号 的文本而使用标题。

# HTML 段落

HTML 段落是通过 <p></p> 来定义的， </p> 是块级元素 。

注：浏览器会自动在段落的前后添加空行。

# HTML 折行

如果你希望在不产生一个新段落的情况下进行换行（新行），请使用 <br/> 标签。

<br/> 标签是一个单体标签，因此只能在开始标签中关闭。

# HTML 链接

HTML 的超链接是通过 <a></a> 来定义的，在 href 属性中指定链接的地址，超链接可以是一个字，一个词，或者一组词，也可以是一幅图像，你可以点击这些内容来跳转到新的文档或者当前文档的某个部分。

超链接的默认样式：

  • 一个为访问过的链接显示为蓝色字体并带有下划线。
  • 访问过的链接显示为紫色字体并带有下划线。
  • 点击链接时，链接显示为红色并带有下划线。
# HTML 链接语法

创建链接的主要 HTML 元素是 <a> （锚）元素。

文本链接：最常见的链接类型是文本链接，它使用 <a> 元素将一段文本转化为可点击的链接。

图像链接：使用图像作为连接，将 <a> 包围住 <img> 元素。

<a> 元素的属性：

  • href ：指定链接目标的 URL 。
  • target ：规定在何处打开目标 URL。仅在存在 href 属性时可用。
    • 可选值：
      • _blank ：新窗口打开。
      • _parent ：在父窗口中打开链接。
      • _self ：默认，当前页面跳转。
      • _top ：在当前窗口打开链接，并替换当前的整个窗体。
      • framename ：在指定的框架中打开被链接文档。
  • download ：指定下载链接。
    • filename ：指定文件名称。
  • hreflang ：指定被链接文档的语言。
    • language_code ：双字符的语言代码。双字符语言代码。
  • media ：规定目标 URL 的媒介类型。默认值： all 。
  • rel ：规定当前文档与目标 URL 之间的关系。
    • index ：文档索引。
    • help ：帮助文档。
    • bookmark ：相关文档。
    • next ：集合中的下一个文档。
    • prev ：集合中的前一个文档。
  • type ：规定目标 URL 的 MIME 类型

# HTML 图像

HTML 图像是通过 <img> 来定义的，图像的名称和尺寸是以属性的形式提供的。

# HTML 水平线

<hr> 标签可用于在 HTML 页面创建水平线。

水平线可用于分割内容。

# HTML 注释

你可以将注释插入到 HTML 代码中，这样可以提高其可读性。

# HTML 文本格式化

HTML 使用标签 <b> 与 <i> 对输出的文本进行格式化。如：粗体 or 斜体

HTML 文本格式化标签：

开始标签 描述 结束标签
<b> 定义粗体文本 </b>
<strong> 定义你要呈现的文本加粗显示 </strong>
<i> 定义斜体文本 </i>
<em> 定义你要呈现的文本斜体显示 </em>
<small> 定义小型文本和旁注 </small>
<sup> 定义上标字 </sup>
<sub> 定义下标字 </sub>
<ins> 定义插入字 </ins>
<del> 定义删除字 </del>

HTML “计算机输出” 标签（短语标签）：

开始标签 描述 结束标签
<code> 定义计算机代码文本 </code>
<kbd> 定义键盘码 </kbd>
<samp> 定义计算机代码样本 </samp>
<pre> 定义预格式文本，被标签包裹的文本会保留空格和换行符 </pre>
<var> 定义变量 </var>

HTML 引文、引用及标签定义：

开始标签 描述 结束标签
<abbr> 定义缩写 </abbr>
<address> 定义地址等信息 </address>
<bdo> 定义文字方向 </bdo>
<blockquote> 定义长的引用 </blockquote>
<q> 定义短的引用 </q>
<cite> 定义引用、引证 </cite>
<dfn> 定义项目 </dfn>

# HTML 元素

开始标签 元素内容 结束标签
<p> 这是一个段落 </p>
<a href="index.html"> 这是一个链接 <a>
<br> 换行

# HTML 元素语法

  • HTML 元素以开始标签起始，以结束标签终止
  • 某些元素具有空内容
  • 大多数 HTML 元素可以拥有属性

# HTML 空元素

没有内容的元素被称为空元素，空元素是在开始标签中关闭的。

<br> 就是没有关闭标签的空元素，在开始标签中添加斜杠 <br/> ，是关闭空元素的正确方法。

注意：HTML 标签对大小写并不敏感，但是更推荐使用小写标签，因为万维网联盟（W3C）在 HTML 4 中推荐使用小写，在 XHTML 中强制使用小写。HTML 属性也是推荐使用小写。

# HTML 属性

属性是 HTML 元素提供的附加信息。

# HTML 属性语法

  • HTML 元素可以设置属性
  • 属性可以在元素中添加附加信息
  • 属性一般描述于开始标签
  • 属性总是以名称 / 值对的形式出现，如： name="value"

# HTML 属性 & 标签参考

标签：HTML 标签列表 | 菜鸟教程

属性：HTML 全局属性 | 菜鸟教程

适用于大部分元素的属性：

属性 描述
class 为 html 元素定义一个或多个类名，类名从样式文件引入 (.css文件)
id 定义元素的唯一 id
style 规定元素的行内样式
title 描述了元素的额外信息（作为工具条使用）
