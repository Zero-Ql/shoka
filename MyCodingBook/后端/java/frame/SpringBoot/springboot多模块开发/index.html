# 多模块开发的坑

由于第一次使用多模块开发，并不熟练，所以踩了很多坑，这里记录一下

  1. 访问后台接口报 404

​ 访问所有接口都报 404，这是由于 spring 没有扫描到你写的 bean 所导致的，一般来说， spring 容器只会扫描启动类所在的包以及其下的子包中的所有类，但是开发不会将所有的模块都放在一个包下面，所以如果你 service 、 dao 等模块放在了启动类所在的包之外，那么 spring 就不会去扫描注册这些 bean

​ 解决方法是在启动类的 @SpringBootApplication 注解上面加上 scanBasePackages={"xxx.xxxxx.*"}

​ xxx.xxxxx 是你所共有的包，就比如说我所有的模块都是以 com.example.xxx 开始。
