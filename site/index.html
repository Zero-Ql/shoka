# 建站日志

  • 2024.6.22

    • 更换 Blog 主题为 Shoka

  • 2024.4.26

    • 由于域名到期，所以网站域名更换为 yunyat.cloud

  • 2023.11.7

    • 大概是 10 月份吧，由于博客访问速度实在过于缓慢（有时甚至断连…），所以将 IP 限制关闭了

  • 2023.9.7

    • 添加了基于 LeanCloud 的访问者统计功能
    • 更改了亮暗主题的切换模式，根据时间来切换， 07：00-19：00 为亮，反之为暗（总之就是添加了，生不生效就不知道辽）
    • 添加了搜索功能，位于博客右上角

  • 2023.5.2 （凌晨三点多…）

    • 添加了评论功能

  • 2023.5.1

    早八点五十，服务器遭遇异常登录，上服务器一看，好家伙，直接被植入挖矿程序了呗， CPU 直接跑满，幸好服务器是两核的，才不至于摊掉，而攻击者登录的账号是 git （自定义账户），可能是我密码设简单了？

    解决方法：直接 kill 掉了进程，暂时没发现定时重启

    痛定思痛，决定换掉 ssh 登录端口和限制国外 IP 访问并且重新设置了一个更复杂的密码（攻击者使用 IP 属于捷克共和国，登录地点位于 Stare Mesto ）

    所以现在本站是只能国内访问咯

  • 2023.4.30

    • 添加了相册功能
    • 添加了友链的随机顺序

  • 2023.4.28

在本月中 (好像又是月初？) 在网上翻关于 get 和 post 请求的时候无意间看到了一个大佬的博客，他是软工专业的，看他的博客我发现了一件事 (或许很早之前就发现了)，真正的技术大部分都在那些大佬的博客中，而这也是我熬到凌晨两点半也要把他的博客看完的原因，现在想想，好像也不亏

  • 为什么建站？

当然是为了分享知识啦

个人博客对于我来说既能记录自己的成长，也能分享一些技术知识，还能当一个云相册用 (当然，由于目前技术不够，暂时还未能实现相册功能🫤)
